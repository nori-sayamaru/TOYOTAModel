TOYOTA-Model ® 4.0
Semantic Safety Architecture inspired by Toyota Production System

本リポジトリは、AIの推論を速くするのではなく、誤った方向へ進みにくくするための設計と実装を示すプロトタイプです。
画像→意味抽出→構造化→判断（停止・徐行・進行）までを一貫して行い、「結論」と同時に「理由の構造」も残すことを目的としています。

What this code actually does

このノートブックは、街中の画像を入力として
①意味抽出（BLIP）→ ②意味要素の蒸留 → ③安全側スコアリング → ④行動判断
を行います。
重要なのは、最終判断だけでなく、どの観測が・どの判断に・どれだけ寄与したかを明示的に保持する点です。

Why this matters (思想が実装に落ちている点)

本実装は、LLMの hallucination を「後から検出する」のではなく、
判断が暴走しにくい構造を先に組むという発想に基づいています。
トヨタ生産方式の「異常があれば止める」「ブラックボックスにしない」という思想を、
スコア分解・理由ツリー・観測ログという形でAIに転写しています。
